Web Technologies

What is a HTML?

A html the standard language for creating webpages.
it is the structure and content of a webpage telling the browser how to display things like names, paragraphs,links and more

What is CSS?

CSS or Cascading Style Sheets, is a stylesheet language used to describe a document in a language similar to HTML.
this divides the imagery and structure of a webpage from its visual appearence and layout.

What is JavaScript?

JavaScript is a very high-level programming language thats normally used to create interactive
or clickable content on web pages.

What is Bootstrap?

Bootstrap in computer terms, is a open-source framework used for developing responsive websites and applications.
these are normally built on HTML, CSS, and JavaScript.
